# Changelog

All notable changes to cligh.nvim will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Full Vim editing capabilities in PR creation form
- Inline markdown editor with syntax highlighting
- Smart branch management (auto-commit and auto-push)
- Real Vim buffer for PR editing instead of input prompts
- Support for all Vim commands, motions, and keybindings
- Custom syntax highlighting for form elements
- `:w` and `Ctrl-s` shortcuts to submit PR
- Automatic detection and handling of uncommitted changes
- Automatic detection and pushing of unpushed branches
- PR listing with floating window interface
- PR viewing with markdown formatting
- PR status checking
- PR checks viewing
- Local PR checkout functionality

## [0.1.0] - 2025-11-05

### Added
- Initial release
- Basic PR creation with GitHub CLI integration
- Floating window UI
- Draft/Ready for Review toggle
- PR listing functionality
- PR details viewing
- PR status checking
- PR checks viewing
- PR checkout functionality
- Complete documentation (README, help docs, contributing guide)
- Example configurations
- MIT License

### Changed
- BREAKING: Redesigned PR creation form with inline Vim editing (from simple input prompts)
